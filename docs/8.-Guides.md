# Harbor Guides

Practical guides for building and operating local AI setups with Harbor. These pages focus on common workflows that combine multiple Harbor services rather than documenting a single service in isolation.

- [Local LLM Stack with Docker Compose](./8.1-Local-LLM-Stack-with-Docker-Compose.md)<br/>
  Build a local LLM stack with Harbor's default Ollama and Open WebUI services, Docker Compose orchestration, and optional cross-service additions such as SearXNG, Speaches, and ComfyUI.
- [Ollama + Open WebUI + SearXNG Local Web RAG Setup](./8.2-Ollama-Open-WebUI-SearXNG-Local-Web-RAG-Setup.md)<br/>
  Add self-hosted web search to the default Ollama and Open WebUI stack with Harbor's SearXNG integration.
