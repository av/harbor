# Harbor Guides

Practical guides for building and operating local AI setups with Harbor. These pages focus on common workflows that combine multiple Harbor services rather than documenting a single service in isolation.

- [Local LLM Stack with Docker Compose](./8.1-Local-LLM-Stack-with-Docker-Compose.md)<br/>
  Build a local LLM stack with Harbor's default Ollama and Open WebUI services, Docker Compose orchestration, and optional cross-service additions such as SearXNG, Speaches, and ComfyUI.
- [Ollama + Open WebUI + SearXNG Local Web RAG Setup](./8.2-Ollama-Open-WebUI-SearXNG-Local-Web-RAG-Setup.md)<br/>
  Add self-hosted web search to the default Ollama and Open WebUI stack with Harbor's SearXNG integration.
- [Run Coding Agents with Local LLMs](./8.3-Run-Coding-Agents-with-Local-LLMs.md)<br/>
  Use Harbor Launch to run Codex, Claude Code, OpenCode, and other host coding tools against local OpenAI-compatible backends.
- [Harbor vs Manual Docker Compose for Local AI](./8.4-Harbor-vs-Manual-Docker-Compose-for-Local-AI.md)<br/>
  Compare a manual Docker Compose local AI stack with Harbor's service selection, cross-service integrations, configuration, CLI/App workflows, Launch adapters, and `harbor eject`.
- [OpenAI-Compatible Local LLM Backends](./8.5-OpenAI-Compatible-Local-LLM-Backends.md)<br/>
  Choose and swap Harbor backends such as Ollama, llama.cpp, vLLM, Docker Model Runner, MLX, and oMLX for local OpenAI-compatible frontends, clients, and Harbor Launch workflows.
- [Self-Hosted AI Stack for Developers](./8.6-Self-Hosted-AI-Stack-for-Developers.md)<br/>
  Plan a private AI stack with Harbor across chat, coding, RAG/search, tools, image generation, voice, workflows, benchmarks, and observability.
- [Run Hermes Agent with Harbor](./8.7-Run-Hermes-Agent-with-Harbor.md)<br/>
  Run Hermes Agent with Harbor local LLM backends through `harbor launch` or the containerized Hermes service/API server.
- [Run MLX on Apple Silicon with Harbor: Local LLMs and HuggingFace Models](./8.8-Run-MLX-on-Apple-Silicon-with-Harbor.md)<br/>
  Install and run Apple's MLX framework on M-series Macs, discover and pull optimized models from HuggingFace (mlx-community), and integrate MLX into full Harbor stacks with Open WebUI, SearXNG, and coding agents.
- [Docker Model Runner Local LLMs with Harbor on Mac, Linux, and Windows](./8.9-Docker-Model-Runner-with-Harbor.md)<br/>
  Use Docker's official Model Runner as a backend on every supported platform. Find GGUF models via `docker model search --source huggingface`, pull with Harbor's unified interface, and run complete local LLM workflows.
