### [MCP Inspector](https://github.com/modelcontextprotocol/inspector)

> Handle: `mcp-inspector`<br/>
> URL: [http://localhost:34781](http://localhost:34781)

MCP Inspector is an interactive developer tool for testing and debugging [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) servers. It provides a web UI to connect to any MCP server, browse its tools and resources, and issue test calls.

---

#### Starting

```bash
# Start MCP Inspector
harbor up mcp-inspector

# Open in browser
harbor open mcp-inspector
```

MCP Inspector exposes two ports:

- **34781** — the web UI
- **34782** — the MCP proxy/client API

#### Usage

After starting the service, open the web UI at [http://localhost:34781](http://localhost:34781). From there you can:

- Connect to any MCP server running inside Harbor by entering its internal URL
- Browse available tools and resources
- Issue test calls and inspect responses

You can also launch MCP Inspector without a persistent container via the `harbor mcp` command:

```bash
# Run MCP Inspector as a one-shot tool (no persistent service)
harbor mcp inspector
```

#### Configuration

```bash
harbor config ls | grep HARBOR_MCP_INSPECTOR

# Port on the host where the MCP Inspector web UI is exposed
HARBOR_MCP_INSPECTOR_HOST_PORT         34781
# Port on the host where the MCP Inspector client API is exposed
HARBOR_MCP_INSPECTOR_CLIENT_HOST_PORT  34782
```

See Harbor's [environment configuration guide](./1.-Harbor-User-Guide#environment-variables) to set arbitrary environment variables for the service.
