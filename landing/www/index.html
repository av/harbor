Run Local LLMs with One Command

Harbor manages your local LLM environment. Pre-configured backends, frontends, and services work together out of the box.

# Start Open WebUI and Ollama harbor up # Add web search and voice chat harbor up searxng speaches
Install Harbor Documentation

Features

Local LLMs

Run LLMs on your machine with minimal configuration. Backends connect to Open WebUI automatically.

harbor up ollama llamacpp vllm

Multiple Inference Engines

Supports vLLM, llama.cpp, TGI, TabbyAPI, Aphrodite, SGLang, KTransformers, and more.

Tool Use (MCP)

Add tools to your LLMs using the MCP ecosystem. Includes web UI for management.

harbor up metamcp mcpo

Image Generation

ComfyUI with FLUX integrates with Open WebUI.

harbor up comfyui

Web Search

SearXNG connects to Perplexica, ChatUI, Morphic, and other services for web-enhanced responses.

harbor up searxng perplexica

Voice Chat

Add speech-to-text and text-to-speech with Speaches (OpenAI-compatible API).

harbor up speaches

LLM Workflows

Build workflows using Dify, n8n, LangFlow, FloWise, or Open WebUI Pipelines.

harbor up dify

Remote Access

Access services from your phone via QR codes, local network, or internet tunneling.

harbor qr
harbor tunnel

Configuration Profiles

Save and switch between configurations for different models and setups.

harbor profile save llama4
harbor profile use default

Command History

Track Harbor commands separately from shell history.

harbor history

LLM Scripting

Script workflows and automate LLM interactions with Harbor Boost.

harbor up boost

Export Configuration

Generate a standalone docker-compose file from your Harbor setup.

harbor eject > compose.yml

Quick Start

1

Install Harbor

# Using npm npm install -g @avcodes/harbor # Or using pip pip install llm-harbor # Or using bash curl -fsSL https://raw.githubusercontent.com/av/harbor/main/install.sh | bash
2

Start Harbor

# Start default services (Open WebUI + Ollama) harbor up # Or start with additional services harbor up searxng speaches comfyui
3

Access Your LLM

# Open in browser harbor open # Get URL with QR code harbor qr

Use harbor help to see all available commands and services.
