Quick Start

1

Install Harbor

# Using npm npm install -g @avcodes/harbor # Or using pip pip install llm-harbor # Or using bash curl -fsSL https://raw.githubusercontent.com/av/harbor/main/install.sh | bash
2

Start Harbor

# Start default services (Open WebUI + Ollama) harbor up # Or start with additional services harbor up searxng speaches comfyui
3

Access Your LLM

# Open in browser harbor open # Get URL with QR code harbor qr
